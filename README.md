# Лабиринт сокровищ

Консольная текстовая игра: исследуйте комнаты, подбирайте предметы, решайте загадки и откройте сундук.

## Установка

```bash
poetry install
```

или

```bash
make install
```

## Запуск

```bash
poetry run project
```

или

```bash
make project
```

## Линтер

```bash
make lint
```
